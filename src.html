class src.utils.TrajectoryWriter(path, run_config, environment_config, online_config, transformer_model_config=None)#

Bases: object

The trajectory writer is responsible for writing trajectories to a file. During each rollout phase, it will collect:

  • the observations

  • the actions

  • the rewards

  • the dones

  • the infos

And store them in a set of lists, indexed by batch b and time t.
